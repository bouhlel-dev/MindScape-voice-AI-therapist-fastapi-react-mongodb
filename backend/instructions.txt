# AI Therapist API

FastAPI backend with MongoDB Atlas for AI therapist application.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Update `.env` with your MongoDB Atlas credentials

4. Run the application:
```bash
uvicorn main:app --reload
```

## API Endpoints

### Users
- `POST /users/` - Create a new user
- `GET /users/{user_id}` - Get user by ID

### Messages
- `POST /messages/` - Create a message
- `GET /messages/user/{user_id}` - Get user messages
- `GET /messages/session/{user_id}/{session_id}` - Get session messages

## Collections

Collections are automatically created on startup:
- **users** - User accounts
- **messages** - Therapy conversation messages
